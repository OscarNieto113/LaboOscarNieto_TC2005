¿Cuáles son las partes de un URL?

  1. Protocolo: Siendo el mas comun http o https
  2. Dominio: Dominio en el que se encuentra alojado
  3. Ruta: ubicacion del recurso en el servidor
  4. Parámetros: datos adicionales, se suele utilizar para hacer consultas

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Obtener datos
  • HEAD: en lugar de mandar todo el mensaje http con head y body, el head solo manda el header
  • POST: enviar enviar informaciónr
  • PUT: actualizar información, solo una parte
  • PATCH: actualiza una parte del recurso
  • DELETE: eliminar recursos
